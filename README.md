# CSV-Megamation
csv-megamation est une site web d'une organisation qui gere l'ecole qui permet aux employes de 
soumettre des demandes d'ordere de travail pour des reparatation et de maintanance

# Technologie utilise  
  # 1. Frontend
        + ReactJs
        + Chakra UI
        + Typescript
        + Axios 
  # 2. Backend
      + Express
      + mongoose
      + Typescript
  # 3. Base de Donnes
      + MongoDB
